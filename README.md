# KyonggiTaxi
# 경택(KyonggiTaxi)

경택은 경기대학교 학생들을 위한 택시 합승 매칭 웹사이트입니다.
사용자는 닉네임을 입력한 뒤 택시팟 방을 만들거나 기존 방에 참여할 수 있으며, 대기실과 채팅방을 통해 탑승 인원을 확인하고 약속을 조율할 수 있습니다.

## 프로젝트 소개

학교 주변에서 같은 방향으로 이동하는 학생들이 택시비를 나누어 부담할 수 있도록 도와주는 웹 서비스입니다.
방 생성자는 출발지, 도착지, 계좌번호, 모집 인원을 입력해 방을 만들고, 참여자는 방 목록에서 원하는 방을 선택해 입장할 수 있습니다.

## 주요 기능

* 닉네임 입력 후 서비스 시작
* 택시팟 방 생성
* 방 이름, 출발지, 도착지, 은행, 계좌번호, 모집 인원 입력
* 카카오 지도 API를 이용한 출발지 위치 선택
* Firebase Firestore를 이용한 실시간 방 목록 관리
* 방 목록에서 기존 방 선택 및 참여
* 모집 인원 초과 시 입장 제한
* 대기실에서 참여자 목록 확인
* 방장의 참여자 승인 기능
* 채팅방에서 메시지 입력 및 정산 관련 대화 가능
* 출발지/도착지 위치 확인
* 방 삭제 및 나가기 기능
* 모바일 화면 중심의 반응형 UI

## 페이지 구성

| 파일명                    | 설명                                          |
| ---------------------- | ------------------------------------------- |
| `startPage.html`       | 닉네임 입력 후 메인 화면으로 이동하는 시작 페이지                |
| `mainPage.html`        | 생성된 택시팟 방 목록을 확인하고 방 생성 페이지로 이동하는 메인 페이지    |
| `roomMakePage.html`    | 방 이름, 출발지, 도착지, 계좌번호, 인원 등을 입력해 방을 생성하는 페이지 |
| `waitingRoomPage.html` | 방 정보와 참여자 목록을 확인하고 채팅방 입장을 준비하는 대기실 페이지     |
| `chatPage.html`        | 참여자들이 대화하고 방 정보를 확인할 수 있는 채팅 페이지            |
| `images/`              | 서비스에 사용되는 이미지 파일 폴더                         |

## 실행 방법

1. 이 저장소를 다운로드하거나 클론합니다.

```bash
git clone https://github.com/사용자이름/KGU-KyonggiTaxi.git
```

2. 프로젝트 폴더로 이동합니다.

```bash
cd KGU-KyonggiTaxi
```

3. 로컬 서버를 실행합니다.

```bash
python -m http.server 5500
```

4. 브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:5500/startPage.html
```

## 사용 흐름

1. 사용자가 `startPage.html`에서 닉네임을 입력합니다.
2. 메인 페이지에서 현재 생성된 택시팟 방 목록을 확인합니다.
3. 방을 새로 만들거나 기존 방을 선택합니다.
4. 방 생성 시 출발지, 도착지, 계좌번호, 모집 인원을 입력합니다.
5. 대기실에서 참여자 목록과 방 정보를 확인합니다.
6. 채팅방으로 이동하여 만남 장소, 시간, 정산 정보를 조율합니다.
7. 필요 시 방장이 방을 삭제하거나 사용자가 방에서 나갈 수 있습니다.

## 주의 사항

* 카카오 지도 API를 사용하려면 Kakao Developers에서 발급받은 JavaScript 키가 필요합니다.
* Firebase Firestore 사용을 위해 Firebase 프로젝트 설정이 필요합니다.
* 브라우저에서 HTML 파일을 직접 여는 것보다 로컬 서버를 실행한 뒤 접속하는 것을 권장합니다.
* 모바일에서 테스트할 경우 PC와 휴대폰이 같은 Wi-Fi에 연결되어 있어야 합니다.

## 프로젝트 목적

경택은 경기대학교 학생들이 보다 안전하고 편리하게 택시를 함께 이용할 수 있도록 돕는 것을 목표로 합니다.
택시비 부담을 줄이고, 같은 방향으로 이동하는 학생들을 빠르게 연결하는 실용적인 웹 서비스를 구현하고자 했습니다.
